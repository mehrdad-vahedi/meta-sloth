# Meta Sloth

### این اولین پروژه اوپن سورس من هست دوستان

قرار هست حدود 2 سال انجامش طول بکشه و یک کتابخانه قدرتمند برای الگوریتم های فراابتکاری با پرفورمنس بالا در نهایت ایجاد خواهد شد

پروژه با `C++` ،درحال توسعه هست ولی هدف اصلی کتابخانه نهایی مخاطبان ریاضی دان هستن که با `python` قطعا راحتترن پس ما هم کتابخونه رو برای پایتون توسعه میدیم.

پروژه 4 فاز داره، الان توی فاز اول هست

همچنین این اولین پروژه من با `CMake` هست و پروژه های قبلی همیشه با `MSVC` انجام شدن
به همین دلیل ممکنه یک مقدار توسعه بیشتر از اونچه که فکر میکنم طول بکشه، اما مطمنم کلی چیز جدید و خفن قراره یاد بگیرم و سعی میکنم مستمر کد زدن برای این پروژه رو رها نکنم.
